# Hoja editable de textos (EU/ES)

Documento preparado para que puedas traducir al euskera/castellano final y sustituir datos reales.

## Como usar esta hoja

- Rellena Texto final EU y Texto final ES donde toque.
- En filas de i18n puedes adaptar tambien nombres, descripciones y datos reales.
- Si una fila no aplica, puedes dejarla vacia o marcarla como OK.
